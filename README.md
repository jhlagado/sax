# ZAX

ZAX is a Node.js-based assembler for the Z80, based on the ZAX language spec (a “structured assembler”).

This repo currently contains scaffolding and type contracts; code generation is not implemented yet.

## Requirements

- Node.js 20+
- Yarn

## Development

- Install: `yarn install`
- Typecheck: `yarn typecheck`
- Test: `yarn test`
- Format: `yarn format`

## Docs

- Spec: `docs/zax-spec.md`
- CLI contract: `docs/zax-cli.md`
- AI team prompt: `docs/zax-ai-team-prompt.md`
